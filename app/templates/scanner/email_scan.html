How to Get Email Headers
Gmail
  1. Open the email
  2. Click ⋮ (three dots)
  3. Select "Show original"
  4. Copy all text
Outlook
  1. Open the email
  2. File → Properties
  3. Copy "Internet headers"
Apple Mail
  1. Select the email
  2. View → Message → All headers
  3. Copy the header block
