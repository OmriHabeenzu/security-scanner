About SecureCheck

An AI-powered multi-vector security scanner built with machine learning

Our Mission

SecureCheck is an AI-powered integrated web security scanner designed to provide comprehensive multi-vector threat detection. Our platform combines advanced machine learning algorithms — trained on real-world datasets — with user-friendly interfaces to make cybersecurity accessible to everyone.

Key Features
  • File Malware Scanner

    PE header analysis using lief + ensemble ML (RF, SVM, Decision Tree) + VirusTotal cross-check across 70+ AV engines.

  • Email Analyzer

    DKIM/SPF/DMARC validation, ML phishing classification trained on SpamAssassin corpus, and live URL scanning from email body.

  • URL Scanner

    Heuristic pattern analysis + VirusTotal & Google Safe Browsing reputation checks for real-time malicious link detection.

  • IP Checker

    AbuseIPDB reputation check (abuse confidence score + report history) with ip-api.com geolocation (city, region, coordinates).

  • Privacy-First

    All ML processing is done locally on our server. File content is never shared externally beyond hash lookups.

Technology Stack
Backend
  • Python & Flask
  • Scikit-learn (RF, SVM, Decision Tree)
  • lief — PE binary analysis
  • SQLite + Flask-SQLAlchemy
  • Flask-Login authentication
Frontend & APIs
  • Bootstrap 5 + Font Awesome 6
  • Inter & Space Grotesk fonts
  • VirusTotal API v3

  • Google Safe Browsing API
